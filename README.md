# Student-data-using-c-++
# 🎓 Student Details Management System (C++)
A simple **Student Details Management System** built using **C++** and **Object-Oriented Programming (OOP)** concepts. This project allows users to enter and display student information through a class-based approach.

## 📌 Features
* Add student details
* Display student information
* Pass/Fail result based on marks
* Demonstrates the use of classes and objects
* Beginner-friendly OOP project

## 🛠️ Technologies Used
* C++
* Object-Oriented Programming (OOP)
* Visual Studio Code / Code::Blocks / Dev-C++ (or any C++ IDE)

## 🚀 Getting Started
### Prerequisites
* A C++ compiler (G++, MinGW, MSVC, etc.)
### Compile the Program
```bash
g++ main.cpp -o student
```
### Run the Program
```bash
./student
```
**On Windows:**
```bash
student.exe
```
